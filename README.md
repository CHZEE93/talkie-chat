# Talkie - 가족 단톡방 웹서비스

**바이브코딩(VibeCoding)**에 의해 구현된 Firebase 기반의 **실시간 가족 채팅 웹 애플리케이션**입니다.

## 주요 기능

- 이메일 기반 회원가입/로그인  
- 초대 코드를 통한 가입 제한  
- 실시간 채팅  
- 사용자별 컬러 자동 지정  
- 이모지 지원  
- 모바일 최적화 UI  

## 사용된 기술 스택

- Next.js 14  
- TypeScript  
- Firebase (Authentication, Firestore)  
- Tailwind CSS  
- React Firebase Hooks  
- Emoji Mart
